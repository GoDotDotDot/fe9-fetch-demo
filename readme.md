# FE9 Fetch Demo

这里是前端九部 fetch 部分示例代码，文章地址 [fetch](https://www.yuque.com/fe9/basic/rfwqry)。

## 如何使用

```shell
$ git clone https://github.com/GoDotDotDot/fe9-fetch-demo.git
$ cd fe9-fetch-demo
$ yarn
$ node server.js
```

如果需要使用跨域版本，需要同时使用 `node cors.js` 命令来启动 cors 服务。
